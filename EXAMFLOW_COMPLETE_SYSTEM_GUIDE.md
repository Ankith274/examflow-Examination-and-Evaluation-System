## 🔐 Security & AI Proctoring

### Features
- ✅ Face detection (tracks student presence)
- ✅ Object detection (prevents cheating)
- ✅ Audio monitoring (detects discussion)
- ✅ Screen recording (monitors activity)
- ✅ Tab switching prevention
- ✅ Full audit trail

### Flagging System
**Current Flags:** 12 attempted cheating cases

1. **No Face Detected (2x)** - Critical
   - Student disappeared from camera
   - Action: Manual review required

2. **Prohibited Item** - Warning
   - Phone or notes detected
   - Action: Document & assess

3. **Multiple People** - Critical
   - More than one person visible
   - Action: Invalidate attempt

### Review Workflow
1. **Access Flagged Records** → View video/screen
2. **Analyze Evidence** → Check timestamps
3. **Make Decision**:
   - ✓ Approve (False alarm)
   - ✗ Reject (Invalid attempt)
   - 🚩 Escalate (Manual review)
4. **Document** findings
5. **Notify** student if needed

## ❓ Frequently Asked Questions

**Q: Can I use multiple roles?**
A: Yes! Log out and select a different role to switch.

**Q: Is data saved locally?**
A: Currently demo data. Full system requires backend database.

**Q: Can I download reports?**
A: Yes! Admin can export as CSV/PDF/Excel.

**Q: What if I forget password?**
A: Use demo credentials or contact support.

**Q: How long does exam take?**
A: Typically 60-120 minutes depending on exam.

**Q: When do I get results?**
A: Within 24 hours after submission.

**Q: Can exams be retaken?**
A: Depends on teacher settings (default: once).

**Q: What about cheating detection?**
A: AI proctor monitors face, objects, audio, screen.

**Q: Can I take exam offline?**
A: No, requires internet connection for proctoring.

**Q: How to report technical issues?**
A: Email: support@klh.edu.in or call help desk.

## 🎓 Best Practices

### For Students
✅ Test camera/mic before exam  
✅ Join exam 5 minutes early  
✅ Have water bottle ready  
✅ Close all other applications  
✅ Use stable internet connection  
❌ Don't wear hats/sunglasses  
❌ Don't look away from screen  
❌ Don't use external help  

### For Teachers
✅ Review questions before publishing  
✅ Test exam settings beforehand  
✅ Provide clear instructions  
✅ Monitor for technical issues  
✅ Grade promptly within 24 hours  
❌ Don't change exam after publishing  
❌ Don't use ambiguous questions  
❌ Don't publicly display scores  

### For Admins
✅ Regular backups  
✅ Monitor system performance  
✅ Review flagged attempts  
✅ Update student records  
✅ Audit access logs  
❌ Don't modify results  
❌ Don't skip security updates  
❌ Don't share admin credentials
